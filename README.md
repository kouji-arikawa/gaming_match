# <ゲーミング・マッチ>

## サイト概要
### サイトテーマ
<ゲームを1人ではなく、複数人で出来るようにするマッチングサイト>

### テーマを選んだ理由
<コロナ過でゲームと出会い、ゲームにあってほしいアプリを作りたい>

### ターゲットユーザ
<ゲームをしている人>

### 主な利用シーン
<１人ではなく、誰かとパーティーを組んでゲームをしたい時>
